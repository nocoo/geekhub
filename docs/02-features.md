## RSS 订阅管理

### 订阅源支持

| 格式 | 支持情况 |
|------|----------|
| RSS 2.0 | ✅ 完整支持 |
| RSS 1.0 | ✅ 完整支持 |
| Atom | ✅ 完整支持 |
| JSON Feed | ⚠️ 计划中 |

### 添加订阅源

1. 点击侧边栏 "+" 按钮
2. 输入 RSS 地址
3. 选择分类（可选）
4. 配置抓取间隔

**支持的输入格式：**
- 标准 RSS/Atom URL
- 网站首页（自动发现 RSS）
- RSSHub 路由语法

### 分类管理

- 创建自定义分类
- 设置分类图标和颜色
- 拖拽排序
- 批量移动订阅源

### 批量操作

- **OPML 导入**：从其他阅读器导入订阅列表
- **OPML 导出**：备份订阅列表
- **批量删除**：选择多个订阅源删除
- **批量移动**：选择多个订阅源移至其他分类

### RSSHub 集成

GeekHub 内置 RSSHub 路由支持，可直接使用 RSSHub 语法：

```
# 示例：微博用户
/weibo/user/1234567890

# 示例：知乎专栏
/zhihu/zhuanlan/example

# 示例：GitHub 仓库
/github/repos/owner/repo
```

配置自定义 RSSHub 实例：

1. 打开设置 → RSSHub
2. 输入实例地址（如 `https://rsshub.app`）
3. 保存配置

## AI 增强功能

### BYOM (Bring Your Own Model)

GeekHub 支持任何 OpenAI 兼容的 API 端点：

| 服务商 | 支持情况 |
|--------|----------|
| OpenAI | ✅ 完整支持 |
| Azure OpenAI | ✅ 完整支持 |
| Anthropic (via proxy) | ✅ 通过代理 |
| 本地 Ollama | ✅ 完整支持 |
| 其他兼容 API | ✅ 通过自定义 Base URL |

**配置方法：**

1. 打开设置 → AI
2. 填写 API Key
3. 设置 Base URL（默认 OpenAI）
4. 选择模型

### 智能摘要

为长文章自动生成摘要：

- 点击文章详情页的 "AI 摘要" 按钮
- 系统调用 AI 生成 2-3 句话的摘要
- 摘要自动缓存，避免重复调用

**摘要策略：**
- 优先从文章 `<description>` 提取
- 若太短则调用 AI 生成
- 支持自定义 Prompt

### 一键翻译

支持将外语文章翻译为中文：

**翻译模式：**
- **手动翻译**：点击翻译按钮
- **自动翻译**：为订阅源开启自动翻译

**翻译范围：**
- 标题翻译
- 正文翻译
- 保留原文对照

**缓存机制：**
- 翻译结果自动缓存
- 同一文章不会重复调用 API
- 支持清除缓存重新翻译

## 阅读体验

### 沉浸式阅读器

ReaderView 提供专注的阅读体验：

- 全屏阅读模式
- 去除广告和干扰元素
- 自动提取正文内容
- 支持图片懒加载

### 主题切换

| 主题 | 说明 |
|------|------|
| 浅色 | 白色背景，适合日间 |
| 深色 | 深灰背景，护眼模式 |
| 跟随系统 | 自动切换 |

### 字体设置

- **字体大小**：14px - 24px 可调
- **行高**：1.5 - 2.0 可调
- **字体族**：系统默认 / 衬线 / 无衬线

### 阅读状态

| 状态 | 说明 | 快捷操作 |
|------|------|----------|
| 未读 | 默认状态 | 自动标记已读 |
| 已读 | 打开后自动标记 | 可手动标回未读 |
| 收藏 | 重要文章 | ⭐ 按钮 |
| 稍后阅读 | 暂时保存 | 🕐 按钮 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `j` / `↓` | 下一篇 |
| `k` / `↑` | 上一篇 |
| `Enter` | 打开文章 |
| `Esc` | 关闭阅读器 |
| `m` | 标记已读/未读 |
| `s` | 收藏/取消收藏 |

## 系统特性

### 智能代理检测

GeekHub 自动检测本地代理，无需手动配置：

**检测顺序：**
1. 环境变量 `HTTP_PROXY` / `HTTPS_PROXY`
2. Clash 默认端口 `7890`
3. Clash Verge 端口 `7891`、`7897`、`7898`
4. V2Ray 端口 `10808`、`10809`

**手动配置：**

1. 打开设置 → 代理
2. 选择代理类型（HTTP/SOCKS5）
3. 输入代理地址和端口
4. 测试连接

### 实时更新 (SSE)

基于 Server-Sent Events 的实时推送：

- **抓取进度**：显示当前抓取状态
- **新文章通知**：有新文章时提示
- **错误报告**：抓取失败时显示原因

### 图片代理

自动处理图片防盗链：

- 代理所有外链图片
- 自动添加正确的 Referer
- 支持懒加载优化

### 数据管理

| 功能 | 说明 |
|------|------|
| 数据统计 | 订阅源数量、文章总数、存储占用 |
| 清理缓存 | 清除翻译缓存、图片缓存 |
| 清理文章 | 删除 N 天前的已读文章 |
| 导出数据 | 导出完整数据备份 |
